Hi, I'm Dave!


DMF Search is a recreation of the user interfaces for Google Search, Image Search, and Advanced Search.
I built this site while working through Harvard's CS50 Web course, project Search.


DMF Search is built with HTML and CSS, and is deployed using https://htmlpreview.github.io/.
